## Standalone shortcuts (classic only)

When `command-framework` is **`classic`**, these work without the `/pex` prefix:

```text
/promote <user> [ladder]
/demote <user> [ladder]
```

The **modern** framework uses **`/pex ladder <ladder> promote|demote <user>`** instead. See [Rank commands](/commands/ranks).
